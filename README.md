🔗 Link Up

Link Up is a campus-based platform that connects university students with short-term side hustles and allows students to offer their own products and services within the campus ecosystem.



🛠 Tech Stack

- HTML
- CSS
- JavaScript (Vanilla JS)

This project is currently a frontend MVP using browser localStorage for data simulation.

🚀 How to Open the Website Locally

Clone the repository:

git clone <your-repo-link>

Open the project folder in VSCode

Install the Live Server extension

Right-click index.html → Click "Open with Live Server"

✏️ How to Edit or Contribute

If you want to make changes:

Pull the latest changes:

git pull origin main

Create your own branch:

git checkout -b your-branch-name

Make your changes.

Commit and push:

git add .
git commit -m "Your message"
git push origin your-branch-name
📌 Notes

This is an MVP version using localStorage.

Clearing browser storage will reset all data.

No backend integration yet.
